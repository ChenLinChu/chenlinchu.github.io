將模板改以 text/x-template 實現。

接續第一篇的內容，要將存在 template 內的內容，寫在另外一個<script>內。

<div id="app">
  <ol>
    <cart-component
      v-for="(item, index) in cart"
      :product="item"
      :key="index"
    ></cart-component>
  </ol>
</div>

<script>
  Vue.component('cart-component', {
    props: ['product'],
    template: `
        <li>
          <h5>{{product.name}}</h5>
          <p>{{product.price}}元
        </li>
    `,
  });

  var app = new Vue({
    el: '#app',
    data: {
      cart: [
        {
          name: 'apple',
          price: 50,
        },
        {
          name: 'banana',
          price: 20,
        },
        {
          name: 'orange',
          price: 30,
        },
      ],
    },
  });
</script>

# 創建另一個 script 標籤：

將這個標籤的 type 設定為 text/x-template，並給予一個 id，這邊命名為 cc，以免與 html 中的標籤混亂。

<script type="text/x-template" id="cc">
  <li>
    <h5>{{product.name}}</h5>
    <p>{{product.price}}元
  </li>
</script>

# 替換 component 內的 template：

將 template 指向 cc 這個 id。

Vue.component('cart-component', {
  props: ['product'],
  template: '#cc',
});

# 資料走向：

  1. Html 內的標籤遍歷 Vue 內的 cart 資料，個別資料為 item。
  2. 創建一個屬性名為 product，用來存放每一筆 item 的資料。
  3. 將 product 資料傳到 component 創建模板。
  4. 找到 id 值為 cc 的模板，將資料傳入使用。
<div id="app">
  <ol>
    <cart-component
      v-for="(item, index) in cart"
      :product="item"
      :key="index"
    ></cart-component>
  </ol>
</div>

<script type="text/x-template" id="cc">
  <li>
    <h5>{{product.name}}</h5>
    <p>{{product.price}}元
  </li>
</script>

<script>
  Vue.component('cart-component', {
    props: ['product'],
    template: '#cc',
  });

  var app = new Vue({
    el: '#app',
    data: {
      cart: [
        {
          name: 'apple',
          price: 50,
        },
        {
          name: 'banana',
          price: 20,
        },
        {
          name: 'orange',
          price: 30,
        },
      ],
    },
  });
</script>
