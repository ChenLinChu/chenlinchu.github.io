Vue安裝方法

Billy Chu   in Kaohsiung
2020-02-15
  • Vue

將 Vue 引入專案中。

# 1. 引入 Vue 的三種方式：

  1. 下載檔案，並以<script>引入

    • 開發版本 - 包含完整的註解等。
    • 生產版本 - 刪除了所有註解並壓縮。

  2. CDN：

    • 官方推薦的最新版本

      <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

    • 也可以引入明確的版本，避免新版本造成不可預期的錯誤。

      <script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>

  3. NPM：

    • 在構建較大型的專案時，可透過 NPM 安裝，搭配打包或模塊等工具使用。

      $ npm install vue

# 2. 安裝瀏覽器擴充工具

官方文件中推薦在瀏覽器安裝開發者工具，這樣可以用更清楚且便利的方式觀察整個專案的進程。我使用的瀏覽器是 chrome，這是官方開發者工具連結。
