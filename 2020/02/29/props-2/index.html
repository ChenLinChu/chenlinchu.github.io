Props-2

Billy Chu   in Kaohsiung
2020-02-29
  • Vue

修正單向數據流所造成的錯誤。

以先前動態綁定圖片連結做範例，另外在模板內增加一個 input 標籤，並給予一個 v-model 監控傳入 photo 模板內的 props 資料。

<div id="app">
  <h2>動態傳遞</h2>
  <photo :img-url="url"></photo>
</div>

<script>
  Vue.component('photo', {
    props: ['imgUrl'],
    template: `
      <div>
        <img :src="imgUrl">
        <input type="text" v-model="imgUrl" />
      </div>
    `,
  });

  var app = new Vue({
    el: '#app',
    data: {
      url: 'https://fakeimg.pl/200x200',
    },
  });
</script>

此時如果將 input 內的資料修改，console 將會跳出錯誤，警告我們不要直接修改 props 傳進來的資料內容。

這時候需要在 photo 元件內加入 data 函式，返回的物件是一個新的 url。

Vue.component('photo', {
  props: ['imgUrl'],
  template: `
    <div>
      <img :src="newUrl || imgUrl">
      <input type="text" v-model="newUrl" />
    </div>
  `,
  data: function() {
    return {
      newUrl: this.imgUrl,
    };
  },
});

這時候就不會出現修改到 Vue 透過 props 傳進來資料的錯誤摟，因為每一個 component 內的資料都是獨立的。
