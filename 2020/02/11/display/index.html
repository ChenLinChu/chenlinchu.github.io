Billy Chu   in Kaohsiung
2020-02-11
  • CSS

紀錄目前遇到所有屬性值的特性。

# block：

  • 區塊元素。
  • 會重新開始新的一行。
  • 可以設定寬高。

# inline：

  • 行內元素。
  • 只占據他們顯示的部分。
  • 無法設定寬高，及上下推擠屬性。

# inline-block：

  • 行內區塊元素。
  • 只占據他們顯示的部分。
  • 可以設定寬高。

# none：

  • 被設置了這個值的元素，會直接消失在頁面。
  • 不佔據任何位置。
  • 如是要讓元素依舊佔據一個位置但看不到，可以用visibility: hidden;。

# flex：

  • 單一方向的排版方式。
  • 可以設定行或列的單一方向。

# grid：

  • 網格排版方式。
  • 可以同時設定行列的分配方式。
