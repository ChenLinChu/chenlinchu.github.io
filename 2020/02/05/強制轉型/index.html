強制轉型。

JavaScript 共有七種型別，分別為：

  1. 原始型別：
    • Number
    • String
    • Boolean
    • Null
    • Undefined
    • Symbol
  2. 物件型別：
    • Object

動態型別的特性，也讓我們在宣告時不需要指定這個變數的型別。

var a = 12345;
console.log(typeof a); //  number

var b = 'Hello';
console.log(typeof b); //  string

var c = true;
console.log(typeof c); //  boolean

型別的轉換分為顯性轉換及飲性轉換。

  1. 顯性：在程式碼中刻意寫出要轉換的型別。
  2. 隱性：沒有明確標明轉型，卻被變了型別。
var a = 42; // number
var b = String(a); // 顯性
var c = a + ''; // 隱性

b; // '42'
c; // '42'
