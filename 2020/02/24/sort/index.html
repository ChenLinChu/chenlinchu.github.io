陣列排序的方法。

# Array.sort()：

預設對一個陣列中的所有元素進行排序，並回傳此陣列，由於此排序方式是依照Unicode 編碼，排序不是穩定的！

let arr = [5, 10, 6, 3, 2, 5, 6, 8, 95];
arr.sort();
console.log(arr); //	[10, 2, 3, 5, 5, 6, 6, 8, 95]

# Array.sort([compareFunction])：

為了擺脫用 Unicode 奇怪的排序方式，我們需要在裡面加上一個比較函式，函式，而內部需放入兩個參數進行比較。

數字升冪排序：

const arr = [5, 10, 6, 3, 2, 5, 6, 8, 95];
arr.sort((a, b) => a - b);
console.log(arr); //	[2, 3, 5, 5, 6, 6, 8, 10, 95]

數字降冪排序：

const arr = [5, 10, 6, 3, 2, 5, 6, 8, 95];
arr.sort((a, b) => b - a);
console.log(arr); //	[95, 10, 8, 6, 6, 5, 5, 3, 2]

compareFunction(a, b)排序的方式會依照以下規則運行：

  1. 如果回傳值小於 0：代表 a 必須要放在較小的索引值，意思是 a 會排在 b 前面。

  2. 如果回傳值等於 0：代表兩個值相等，則 a 與 b 皆不會改變彼此的順序。

  3. 如果回傳值大於 0：代表 a 必須放在較大的索引值，則 a 會排在 b 後面。

# 物件陣列排序：

當陣列中的每一個元素都是一個物件時，根據物件內的某一個值進行排序。

const products = [
  {
    name: '一度讚牛肉麵',
    price: 50,
    expireDate: 300,
  },
  {
    name: '維力炸醬麵',
    price: 30,
    expireDate: 100,
  },
  {
    name: '地獄辣椒乾麵',
    price: 28,
    expireDate: 150,
  },
  {
    name: '科學麵',
    price: 10,
    expireDate: 500,
  },
  {
    name: '隨緣素食牛肉麵',
    price: 80,
    expireDate: 70,
  },
];
products.sort((a, b) => a.price - b.price);
console.log(...products);

// 輸出結果
// {name: "科學麵", price: 10, expireDate: 500}
// {name: "地獄辣椒乾麵", price: 28, expireDate: 150}
// {name: "維力炸醬麵", price: 30, expireDate: 100}
// {name: "一度讚牛肉麵", price: 50, expireDate: 300}
// {name: "隨緣素食牛肉麵", price: 80, expireDate: 70}

上述comparefunction會依照每個物件內price的值進行比較回傳。
