使用 v-bind、v-for 組建 Vue 的元件。

# 首先建立 Vue 的資料內容：

這三筆物件資料會用在後續<li>渲染上。

var app = new Vue({
  el: '#app',
  data: {
    cart: [
      {
        name: 'apple',
        price: 50,
      },
      {
        name: 'banana',
        price: 20,
      },
      {
        name: 'orange',
        price: 30,
      },
    ],
  },
});

# 創建頁面標籤：

  1. 首先創建一個自定義的標籤名稱，這邊叫做<cart-component>。
  2. 在 html 中，元件標籤名稱如為兩個單字，需以 - 號分格。
  3. 用v-for遍歷 cart 內的資料。
  4. 用v-bind為這個標籤設定一個屬性叫product，值為 cart 裡面的 item。
<div id="app">
  <ol>
    <cart-component
      v-for="(item, index) in cart"
      :product="item"
      :key="index"
    ></cart-component>
  </ol>
</div>

# 創建元件：

  1. cartComponent 對應 html 中的 cart-component，在 js 中以小駝峰的寫法也可對應的到，當然如果要沿用標籤的方式 cart-component 也是可以的。
  2. props 代表 html 標籤中 product 的值。
  3. Template 是以字串方式呈現此標籤的內容，也是可以用反斜線的樣板字串，串接所有的內容。
Vue.component('cartComponent', {
  props: ['product'],
  template: '<li>{{product.name}} - {{product.price}}元</li>',
});

或者

Vue.component('cart-component', {
  props: ['product'],
  template: `
      <li>
        <h5>{{product.name}}</h5>
        <p>{{product.price}}元
      </li>
  `,
});

接著我們就可以看到資料在頁面上順利被呈現出來拉！！！
