元件插槽。

每一個元件都有一個基本的框架，如果我們要在這個框架中加入自定義的內容，像下面直接將內容插入在 html 標籤內的話，是沒有效果的。

<div id="app">
  <article-content>
    <p>這是我要插入的文字</p>
  </article-content>
</div>

<script>
  Vue.component('articleContent', {
    template: `
      <div>
        <p>我是一段文章的開頭</p>
        <p>我是一段文章的結尾</p>
      </div>
    `,
  });

  var app = new Vue({
    el: '#app',
  });
</script>

# <slot></slot>：

我們要在模板文字內加入<slot></slot>的標籤，隨著標籤放置的位置，插入的文字將會自動對應該處。

Vue.component('articleContent', {
  template: `
    <div>
      <slot></slot>
      <p>我是一段文章的開頭</p>
      <slot></slot>
      <p>我是一段文章的結尾</p>
      <slot></slot>
    </div>
  `,
});

# <slot>預設文字</slot>：

slot 標籤內也可以加入預設的文字，當該元件未插入任何內容時會顯示出來。

Vue.component('articleContent', {
  template: `
    <div>
      <p>我是一段文章的開頭</p>
      <slot>預設文章內容</slot>
      <p>我是一段文章的結尾</p>
    </div>
  `,
});

# 多個插入的內容與多個<slot>的對應方式：

如下範例共有三個 html 的標籤要插入元件內，代表著我們元件模板內也會有三個<slot>標籤，對應的方式為：

# html 標籤設定：

  • slot="名稱"：html 標籤內需有一個 slot 屬性，並為其命名。

# component 模板設定：

  • name="名稱"：模板內的<slot>標籤需有一個 name 的屬性，並連結到 html 標籤 slot 屬性的值。
<div id="app">
  <article-content>
    <nav slot="nav">網站導覽列</nav>
    <main slot="main">文章主要內容</main>
    <footer slot="footer">網站底部版權聲明</footer>
  </article-content>
</div>

<script>
  Vue.component('articleContent', {
    template: `
      <div>
        <p>我是一個網站的開頭</p>
        <slot name="nav"></slot>
        <slot name="main"></slot>
        <p>聯繫方式</p>
        <slot name="footer"></slot>
      </div>
    `,
  });

  var app = new Vue({
    el: '#app',
  });
</script>
