選擇器

Billy Chu   in Kaohsiung
2020-01-23
  • CSS

我最常使用到的選擇器（不包含偽類選擇器）。

除了基本的id、class選擇器外，以下是我幾乎每個專案的練習都會使用到的。

# 1. *

* {
  margin: 0;
  padding: 0;
}

星號能指向頁面所有的元素，因頁面創造時，元素會有默認的margin及padding值，需先將將他們歸零後，再重新賦值。

#root * {
  color: red;
}

這個星號也能讓你一次選取指定元素下的所有元素。

# 2. X Y

li a {
  text-decoration: none;
}

透過這樣的方式，能選取頁面內所有存在於<li>下的<a>元素。

# 3. X > Y

li > a {
  text-decoration: none;
}

與X Y{}不同的地方在於，它並不會只向所有的元素，以上述例子來看，指向的只有位於<li>底下第一層的<a>元素，然而<a>元素裡面如有其他的<a>元素，將不會受到影響。

# 4. X ~ Y

p ~ span {
  color: red;
}

波浪符號影響的是跟在<p>標籤後方所有的<span>字體顏色皆會變成紅色。

# 5. X + Y

p + span {
  color: red;
}

這個選擇器會影響的範圍，僅限於黏在X後面第一層的Y，透過上面的例子可以看到，只有相鄰在<p>標籤後面的<span>字體顏色會是紅色。

# 6. X[title]

a[title] {
  text-decoration: none;
}

此為屬性選擇器，選擇所有有[title]這個屬性的<a>標籤。

# 7. X[href="demo"]

a[href="https://google.com"] {
  text-decoration: none;
}

如果需要更精確的屬性選擇器，可以加上屬性的值，避免影響到其他的標籤。

# 8. X[href*="demo"]

a[href*='com'] {
  color: red;
}

星號表示所有<a>標籤的href=" "內只要含有 com 字元的，都會被選擇到。

# 9. X[href^="http"]

a[href^='http'] {
  color: yellow;
}

^這個符號表示一個字串的開頭，上述範例表示，如遇到開頭含有 http 的<a>標籤皆會選擇到。

# 10. X[href$=".jpg"]

a[href$='.jpg'] {
  color: blue;
}

$這個符號表示一個字串的結尾，上述範例表示，如遇到結尾含有.jps 的<a>標籤皆會選擇到。

# 11. X[data-*="demo"]

如果一個網頁上同時有許多不同檔案類型的圖片，我們會需要創造多個選擇器，這樣的寫法也極為痛苦。

a[href$='.jpg'],
a[href$='.png'],
a[href$='.gif'] {
  color: yellow;
}

因此我們可以幫這些元素在編寫html時自定義屬性。

<a href="path/to/image.jpg" data-filetype="image"> Image Link </a>
<a href="path/to/image.png" data-filetype="image"> Image Link </a>
<a href="path/to/image.gif" data-filetype="image"> Image Link </a>

接著我們就能透過這個屬性，一次選擇所有的圖片標籤了。

a[data-type='image'] {
  color: red;
}

# 12. X[data~="demo"]

<a href="path/to/image.jpg" data-filetype="image"> Image Link </a>
<a href="path/to/image.png" data-filetype="image"> Image Link </a>
<a href="path/to/image.gif" data-filetype="image mark"> Image Link </a>

當我們要指定的屬性有多個值的話，可以用這樣的方式。

a[data-filetype~='image'] {
  color: red;
}

a[data-filetype~='mark'] {
  border: 1px solid black;
}
