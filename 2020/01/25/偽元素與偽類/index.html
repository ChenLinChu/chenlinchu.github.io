偽元素與偽類

Billy Chu   in Kaohsiung
2020-01-25
  • CSS

偽元素及偽類的差別、用法。

# 先來講講它們的差別

有時候會看到CSS的檔案內有雙冒號::及單冒號:，雙冒號代表的是偽元素，單冒號代表的是偽類，那他們的差別在哪裡呢？

  1. 偽元素::：創造一個全新的假元素，例如::before能在元素前創造一個假的內容。
  2. 偽類:：選擇元素已經存在的狀態，例如a:hover就是在選擇這個<a>標籤滑入時所要執行的語句。

W3C 的官網分別有對偽元素及偽類有詳細的列表個別介紹，以下僅會將所有的方法條列出來。

# 1. 偽元素

Selector Example Example description
::after p::after Insert something after the content of each <p> element
::before p::before Insert something before the content of each <p> element
::first-letter p::first-letter Selects the first letter of each <p> element
::first-line p::first-line Selects the first line of each <p> element
::selection p::selection Selects the portion of an element that is selected by a user
